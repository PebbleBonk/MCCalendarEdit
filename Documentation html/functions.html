editTags.parseMyCoursesCalendar(icalendarFile, outputFileName)[source]¶

Edits MyCourses exported .ics calendar into nicer form

Script takes the .ics file and edits the summary of each event. As MyCourses exported calendar events’ summaries contain extra information in addition to the name of the event, it is split into respective fields supported by the icalendar format, such as location, URL and geo.

The format of the summary and how to split it was deduced based on events exported for five courses so some variation and bugs (information in wrong field/ missing information) might be present in the created calendar.

Parameters:
  • icalendarFile (string) – the calendar exported from MyCourse in icalendar format (.ics) as a path to the file
  • outputFileName (string) – the path and name of the file to which the edited calendar is exported.
Returns:

Nothing
